# U-Projects 项目管理

U-Projects 是一个现代化的项目管理应用，帮助您管理项目信息、任务分配和进度追踪。

## 功能特性

- **数据持久化** - 使用 LocalStorage 本地存储数据
- **响应式设计** - 完美适配桌面和移动设备
- **明亮清新** - 现代化界面设计

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS 3.4
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **图标库**: Lucide React
