# U-Music 音乐播放器

U-Music 是一个现代化的音乐播放器应用，帮助您轻松管理音乐库和播放列表，支持歌词显示和收藏管理。

## 功能特性

- **数据持久化** - 使用 LocalStorage 本地存储数据
- **响应式设计** - 完美适配桌面和移动设备
- **明亮清新** - 现代化界面设计

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS 3.4
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **图标库**: Lucide React
