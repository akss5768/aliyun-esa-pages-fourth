# U-Passwords 密码管理

U-Passwords 是一个安全的密码管理应用，帮助您安全管理您的账户密码和敏感信息。

## 功能特性

- **数据持久化** - 使用 LocalStorage 本地存储数据
- **响应式设计** - 完美适配桌面和移动设备
- **明亮清新** - 现代化界面设计

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS 3.4
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **图标库**: Lucide React
